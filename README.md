# jasm
Road to building an x86 Assembler in C.

Dependencies:

- [Visual Studio Build Tools 2019 (Clang Support)](https://visualstudio.microsoft.com/downloads/)
- [GNUMake (gmake)](http://gnuwin32.sourceforge.net/packages/make.htm)

Compile and run tests using the x64 Native Tools for Command Prompt.

Compile:
```
make all
```

Tests:
```
make test
```
